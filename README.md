## 🌟 Key Features

### 🧮 Triple-Mode Mathematics Engine
- **Standard Mode:** Streamlined graphical layout for fast, everyday calculations.
- **Scientific Mode:** Advanced drop-down functions including Trigonometry (Sin, Cos, Tan), Logarithms, Roots, and Exponents.
- **Bi-Directional Conversions:** Instantly convert Length, Weight, Area, and Temperature with a dedicated custom keypad and auto-updating UI.

### 🌌 Antigravity Particle Galaxy
Hover your mouse over the background to see true fluid physics!
- **Orbital Swirls:** Thousands of perfectly mapped dashes continuously orbiting a galactic center.
- **Magnetic Dodging:** Particles aggressively repel from your mouse cursor using elastic spring physics and smoothly warp back to their rigid orbit track.
- **Kinetic Shockwaves:** Click *anywhere* to send out massive kinetic displacement shockwaves that visibly push the particles out of rotation!

### 💻 Active Virtual Cursor
- Forget standard non-interactive displays! The custom Javascript input engine permits **moving backward and forward** using your keyboard's `Left` and `Right` arrow keys.
- **Click-to-Target:** Simply tap anywhere inside a long number to instantly split your digits and drop the blinking cursor in the middle.
- **Mid-String Edits:** Inject digits or perfectly backspace digits seamlessly from inside a giant number!
- **Auto-Scrolling:** Overflowing text actively tracks the cursor organically just like a native IDE text editor.

### 🎨 Stunning 3D Aesthetics
- Immersive Glassmorphism and Deep 3D CSS button rendering.
- Perfectly mapped and animated physical hardware drop shadows reacting to every button press.
- True Zero-Tint Black & Grayscale Dark Theme perfectly contrasted alongside a vivid Light Theme!

## 🚀 How to Run

Absolutely NO frameworks or local installations required! Built entirely in pure Vanilla HTML, CSS, and JS.
1. Save the project folder anywhere.
2. Double click on `index.html` to instantly open the application inside any modern browser (Chrome, Edge, Firefox, Safari).
